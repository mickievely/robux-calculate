# 로벅스 계산 봇

Discord 서버에서 로벅스(Robux) 재고 관리 및 가격 계산을 자동화하는 Discord 봇입니다.

## 주요 기능

- **재고 관리**: 로벅스 재고를 추가/제거하고 실시간으로 확인
- **가격 설정**: 서버별로 가격 단위와 로벅스 개수를 설정
- **계산기**: 원화 ↔ 로벅스 자동 변환 계산
- **실시간 재고 표시**: 지정된 채널에 실시간 재고 정보를 표시하는 임베드
- **서버별 설정**: 각 서버마다 독립적인 설정 저장

## 설치 방법

### 1. 필수 요구사항

- Python 3.8 이상
- discord.py 라이브러리

### 2. 패키지 설치

```bash
pip install discord.py
```

### 3. 봇 토큰 설정

`로벅스계산봇.py` 파일을 열고 상단의 `BOT_TOKEN` 변수를 수정하세요:

```python
BOT_TOKEN = '여기에_봇_토큰_입력'
```

### 4. 봇 실행

```bash
python 로벅스계산봇.py
```

## 명령어 목록

### 관리자 전용 명령어

- `/재고관리` - 로벅스 재고를 추가하거나 제거합니다
- `/로벅스재고관리` - 로벅스 재고를 추가합니다
- `/로벅스가격관리` - 가격 단위당 로벅스 개수를 설정합니다
- `/로벅스단위` - 가격 단위와 로벅스 개수를 설정합니다
- `/가격관리` - 가격 단위와 로벅스 개수를 한 번에 설정합니다
- `/로벅스실시간재고채널` - 실시간 재고를 표시할 채널을 설정합니다

### 일반 사용자 명령어

- `/로벅스재고` - 현재 로벅스 재고를 확인합니다
- `/로벅스계산기` - 원하는 금액의 로벅스 개수를 계산합니다

## 사용 방법

1. 봇을 서버에 초대합니다
2. 관리자 권한으로 `/로벅스단위` 명령어를 사용하여 가격 단위와 로벅스 개수를 설정합니다
3. `/로벅스실시간재고채널` 명령어로 실시간 재고를 표시할 채널을 설정합니다
4. 실시간 재고 임베드의 버튼을 사용하여 원화 ↔ 로벅스 변환 계산을 할 수 있습니다

## 데이터 저장

봇은 `guild_data.json` 파일에 서버별 설정을 저장합니다. 봇을 재시작해도 설정이 유지됩니다.

## 주의사항

- 관리자 권한이 필요한 명령어는 서버 관리자만 사용할 수 있습니다
- 각 서버의 설정은 독립적으로 관리됩니다
- 봇이 재시작되어도 버튼과 실시간 재고 표시는 계속 작동합니다

## 기술 스택

- Python 3.8+
- discord.py
- JSON (데이터 저장)
